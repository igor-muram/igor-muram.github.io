Welcome to Web Template!


Автор шаблона: Мурамщиков Игорь

Web Template - это стартовый шаблон для быстрого старта вёрстки. Шаблон включает в себя наиболее часто используемые мной инструменты, такие как:

  • Bootstrap 4 (grid и reboot)
  • Gulp 4
  • Sass и Scss
  • Browsersync
  • Autoprefixer
  • Clean-CSS
  • Rsync
  • Bower
  • Font Awesome
  • JQuery
  • Плагины mmenu и css-hamburgers

Кроссбраузерная совместимость шаблона: IE9+.

Шаблон включает в себя файл .htaccess, содержащий правила кэширования для веб-сервера.

По умолчанию в шаблоне созданы:

  • Меню
  • Прелоадер
  • Кнопка возврата к началу страницы
  • Иконки социальных сетей

Как использовать шаблон

  1. Загрузите Web Template со страницы GitHub.
  2. Установите Node Modules командой npm i.
  3. Запустите шаблон командой gulp.

Как взаимодействовать с шаблоном при помощи Gulp

  • В начале файла gulpfile.js можно изменить желаемый css-препроцессор (sass или scss).
  • gulp: запуск задачи 'default', содержащей sass|scss, js, watch и browserSync для веб-разработки.
  • build: сборка проекта и обновление папки dist.
  • rsync: деплой проекта на сервер из папки dist с помощью Rsync.

Правила работы с шаблоном

  1. Все HTML-файлы должны иметь одинаковое начальное содержимое, как в файле app/index.html.
  2. Комментарий Template Basic Images Start в файле app/index.html: начало стандартных пользовательских картинок (og:image для социальных сетей, favicons - иконки для различных устройств).
  3. Комментарий Custom Browsers Color Start в файле app/index.html: установка цвета заголовка браузера на различных устройствах.
  4. Комментарий Custom HTML в файле app/index.html: начало пользовательской HTML-разметки.
  5. Для установки новой JQuery библиотеки используйте команду "bower i имя_библиотеки" в окне команд. Библиотеки автоматически добавляются в папку app/libs. Для работы команды необходимо установить Bower (команда "npm i -g bower"). После установки пропишите путь к js файлу библиотеки в файле gulpfile.js в задачу 'scripts'.
  6. Все пользовательские скрипты должны находиться в файле app/js/common.js.
  7. Все Sass|Scss переменные должны находиться в файле app/sass/_vars.sass | app/scss/_vars.scss.
  8. Все медиа-запросы Bootstrap должны находиться в файле app/sass/_media.sass | app/scss/_media.scss.
  9. Все CSS файлы подключаемых библиотек должны находиться в файле app/sass/_libs.sass | app/scss/_libs.scss.
